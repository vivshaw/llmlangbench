# Task: Add Two Numbers

## Description

Implement a function called `add` that takes two numbers and returns their sum.

## Requirements

- The function should accept two numeric arguments
- The function should return their sum
- Handle negative numbers correctly
- Handle floating point numbers correctly

## Examples

```
add(1, 2)    // => 3
add(-1, 1)   // => 0
add(0.1, 0.2) // => 0.30000000000000004 (standard float behavior is fine)
```
